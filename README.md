# Bitcoin (BTC) Price Prediction

## Problem Statement
Predict the future price of Bitcoin using historical cryptocurrency market data and machine learning techniques.

## Dataset
Historical Bitcoin price data including:
- Open price
- Close price
- High and low values
- Trading volume
(Data sourced from public cryptocurrency datasets)

## Approach
- Data cleaning and preprocessing
- Time series analysis
- Feature engineering
- Model training and evaluation

## Models Used
- Linear Regression
- Random Forest Regressor

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Results
The model captures historical price trends and predicts Bitcoin prices with reasonable accuracy.

## Future Improvements
- Use LSTM / deep learning models
- Add technical indicators (RSI, MACD)
- Improve time-series forecasting
